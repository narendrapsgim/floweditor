How do I add nodes?

Drag nodes to the workspace to make it appear.

How do I delete nodes?

1. Select the node you want to delete by clicking its name.
2. Click the Delete key.


What do the different colors mean?

The color of the nodes represent the type of Node, There are three types of Nodes: Input, Processing and Output.

The color of the connection circles represent the data type:
Green = Number
Purple = Any
Yellow = Trigger
Blue = Toggle
